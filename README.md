# marvels-app

## Project setup

```
npm install
```

## Copy the .env.example file to .env.development

```
cp .env.example .env.development
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Compile TailwindCss

```
npm run tailwind
```
